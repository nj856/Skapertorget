# SkaperTorget - Help API - Aleksandra
 
**Ressurs:** Hjelpeforespørsler / Support Tickets

---

## Beskrivelse

Dette er et REST API for håndtering av hjelpeforespørsler i SkaperTorget. API-et støtter full CRUD (Create, Read, Update, Delete) og er bygget med Django REST Framework

---

## Installasjon og kjøring:

### Forutsetninger
- Python 3.12 eller nyere
- Django 5.2+
- Django REST Framework
